# ui-button-icon
A button icon with an action feedback API

#Styles
```css
--color: #FFF;
--background-color: gray;
--hover-color: #FFF;
--hover-background-color: #1c7ef8;
--disabled-color: #C4C5D5;
--success-color: #3DBEA3;
--error-color: #F1536E;
```

#Actions
```js
#.loading(); //Click default
#.success();
#.error();
#.init(); //back to normal
```
